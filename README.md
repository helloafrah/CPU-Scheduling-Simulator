# CPU Scheduling Simulator

## 📌 Objective
This project demonstrates CPU Scheduling Algorithms including FCFS, SJF, and Round Robin.

## ⚙️ Algorithms Implemented
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Round Robin (RR)

## 🧩 Inputs
- Process ID
- Arrival Time
- Burst Time

## 📊 Outputs
- Waiting Time
- Turnaround Time
- Gantt Chart (visual representation)

## 💻 Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Run
1. Open index.html in browser
2. Enter process details
3. Click on any algorithm button
